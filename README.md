#  Excel Dashboard Project

##  Project Overview

This project is an Excel-based dashboard that analyzes data using charts, formulas, and pivot tables.
It helps in understanding trends, performance, and key insights from the dataset.

---

##  Files Included

* Excel File (.xlsx)
* Dashboard Screenshots
* Dataset 
 
---

##  Key Insights

* Sales performance analysis
* Category-wise comparison
* Trend analysis (monthly/yearly)
* Key metrics and KPIs

---

##  Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Formulas (SUM, IF, VLOOKUP, etc.)
* Data Cleaning

---

##  Features

* Interactive dashboard
* Dynamic charts
* Easy-to-understand layout
